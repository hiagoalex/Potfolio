No Capricho

Desenvolvi o sistema completo e o cardápio digital para o No Capricho, um restaurante especializado em esfihas.

Funcionalidades

  • Gestão de Pedidos
  • Controle de Estoque
  • Cardápio Digital Interativo
  • Integração com Sistemas de Pagamento

Tecnologias Utilizadas

  • Node.js para o backend
  • Angular para o frontend
  • HTML, CSS e JavaScript para o cardápio digital

Impacto

Melhorou a eficiência do processo de pedido e gestão de estoque, oferecendo uma experiência de cliente moderna e fácil de usar.
